# Bookmarks (convoro-bookmarks)

A first-party Convoro extension that lets members **save posts to a personal
reading list**.

A bookmark toggle appears on every post; saved posts collect on a private
`/bookmarks` page (linked from the header when you're signed in). Opt-in —
install it only if your community wants it.

Adds a `bookmarks` table (removed on uninstall). MIT licensed.
